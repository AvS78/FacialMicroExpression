Environment File
facial_micro_expression.yml

1. Run Final_HOF_Full_Face.ipynb to extract the HOF features of full face and save features
2. Run Final_HOF_Crop_Nose_Mouth_Extraction.ipynb to extract the HOF for cropped face
and eyes and nose/mouth separately and save features
3. Run Final_LBPTOP_Full Face_Feature_Extraction to extract the LBP-TOP features of the full face and save features
4. Run Final_LBP_TOP_Eye and Nose_mouth extraction to extract the LBP-TOP features of the eye and nose mouth regions and save features
5. Run Final_Classification_Models.ipynb and use the previously extracted features for prediction
